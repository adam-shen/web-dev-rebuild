# Web Development Rebuild

A structured learning repository for rebuilding web development fundamentals from scratch.

## Philosophy

- **No copying from tutorials** - Write every line of code yourself
- **Code from scratch** - Implement solutions based on understanding, not memorization
- **AI as helper, not solver** - Use AI assistance only after attempting problems yourself

## Structure

| Folder | Topic |
|--------|-------|
| `01-html/` | HTML fundamentals |
| `02-css/` | CSS styling and layout |
| `03-javascript/` | JavaScript programming |
| `04-dom/` | DOM manipulation |
| `05-projects/` | Build complete projects |
| `notes/` | Personal learning notes |

## How to Use

1. Start with `01-html/` and work through sequentially
2. In each folder, read the README and explore the notes
3. Attempt challenges before looking at any solutions
4. Document your learning in the notes.md files

## Progress

Track your progress as you complete each section.
